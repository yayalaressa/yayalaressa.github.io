Solusi Build Tool 31.0.0 Is Missing DX Pada Cordova

Rabu, 21 Juli 2021

Hallo teman - teman pembaca blog saya,

Sesuai judul pada postingan ini, saya akan menuliskan bagaimana cara saya mengatasi Build-tool 31.0.0 is missing DX at yang terjadi ketika Cordova mengalami kegagalan build aplikasi.

Mungkin ada teman - teman yang mengalami hal yang sama seperti yang saya alami, namun belum menemukan atau sedang mencari solusinya?

Nah barangkali postingan ini dapat membantu teman - teman, oke langsung saja.

Pada gambar diatas terlihat error yang saya alami.

Apabila pesan error pada Terminal teman - teman sama dengan gambar diatas, maka ikuti langkah berikutnya.

Teman - teman bisa membuka folder tempat SDK Android teman - teman Android/Sdk/build-tools/31.0.0/

Karena saya menggunakan Linux maka yang saya lakukan ialah dengan mencopy file d8 lalu merename hasil copy an d8 menjadi dx

Kemudian buka folder lib lakukan hal yang sama seperti diatas dengan mencopy file d8.jar lalu merename hasil copy an d8.jar menjadi dx.jar

Jika sudah maka silahkan lakukan build ulang aplikasi teman - teman, seharusnya sekarang sudah berhasil.

Namun jika teman - teman masih mengalami kendala, silahkan tanyakan pada kolom komentar dibawah ini.

Sekian dulu postingan kali ini, semoga bermanfaat.
